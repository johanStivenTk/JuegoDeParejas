# Proyecto Plantilla Moderno

Este es un proyecto base creado con HTML5, CSS3 y JavaScript Vanilla

## Características

- **Diseño**: mi fuerte no es el diseño, pero he intentado hacerlo lo mas bonito posible
- **Responsivo**: Adaptable a dispositivos móviles y de escritorio.
- **Estructura Limpia**: Código organizado y comentado en español.
- **Sin Dependencias**: Funciona directamente en el navegador sin necesidad de compilación compleja.

## Estructura del Proyecto

```
/
├── index.html          # Estructura principal
├── README.md           # Documentación
└── Assets/             # Recursos del proyecto
    ├── css/
    │   └── styles.css  # Estilos y variables CSS
    ├── js/
    │   └── juego.js    # Lógica de interacción
    ├── img/            # Imágenes de las cartas
    ├── sounds/         # Efectos de sonido
    └── fonts/          # Fuentes personalizadas (Inter)
```

## Cómo Empezar

1. clonar este repositorio o descarga los archivos.
2. abre el archivo `index.html` en tu navegador web favorito.
3. ¡Listo! Deberías ver la página del juego.

## Personalización

- **Colores**: Edita las variables CSS en `:root` dentro de `styles.css`.
- **Contenido**: Modifica el HTML en `index.html`.
- **Lógica**: Agrega tus funciones en `juego.js`.
- **Imagenes**: Reemplaza las imágenes en la carpeta `img`.
- **Sonido**: Reemplaza los sonidos en la carpeta `sounds`.
- **Fuentes**: Reemplaza las fuentes en la carpeta `fonts`.
