# Proyecto Plantilla Moderno

Este es un proyecto base creado con HTML5, CSS3 y JavaScript Vanilla, diseñado con una estética moderna y "premium".

## Características

- **Diseño Moderno**: Uso de glassmorphism, gradientes y tipografía Inter.
- **Responsivo**: Adaptable a dispositivos móviles y de escritorio.
- **Estructura Limpia**: Código organizado y comentado en español.
- **Sin Dependencias**: Funciona directamente en el navegador sin necesidad de compilación compleja.

## Estructura del Proyecto

```
/
├── index.html      # Estructura principal
├── style.css       # Estilos y variables CSS
├── script.js       # Lógica de interacción
└── README.md       # Documentación
```

## Cómo Empezar

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en tu navegador web favorito.
3. ¡Listo! Deberías ver la página de inicio.

## Personalización

- **Colores**: Edita las variables CSS en `:root` dentro de `style.css`.
- **Contenido**: Modifica el HTML en `index.html`.
- **Lógica**: Agrega tus funciones en `script.js`.
